# Open Threads

Questions, ideas, or loose ends without a home yet — things worth
remembering but not settled enough for a project note or a decision.
Remove an item once it's resolved (turned into a decision, filed under a
project, or dropped) instead of marking it done and leaving it here; this
file should only ever hold what's still actually open.
